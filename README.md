# Project Dawn 🌅

> **The first million. Then the life I choose.**

Project Dawn 是一款专注于**长期储蓄目标管理**的个人 Web App，旨在帮助用户记录存款、追踪目标进度，并通过数据可视化和智能预测建立长期储蓄习惯。

不同于传统记账软件，Project Dawn 不关注日常消费，而是围绕一个明确的长期目标（默认 100 万元）构建完整的目标管理体验，让每一次存款都能直观地转化为距离梦想更近的一步。

# 🎯 功能特性

## 💰 存款记录

* 新增存款记录
* 删除存款记录
* 日期、金额、备注管理
* 自动累计存款

---

## 📊 Dashboard

实时展示：

* 当前累计存款
* 距离目标剩余金额
* 当前完成率
* 动态目标进度条

---

## 📈 数据统计

自动计算：

* 本月新增存款
* 今年新增存款
* 最大单笔存款
* 平均每月新增存款

---

## 🎯 Goal System

支持：

* 默认目标：¥1,000,000
* 自定义目标金额
* 自动更新所有统计数据
* 自动计算完成率

---

## 🔮 Timeline Forecast

根据历史数据预测：

* 平均每月存款
* 剩余所需月份
* 预计完成日期

若数据不足，将显示：

> Start saving to unlock your timeline.

---

## 📊 Savings Trend

按月份统计：

* YYYY-MM 汇总
* 动态柱状图
* 自动更新趋势

---

## 🏆 Milestones

自动生成：

* 10%
* 25%
* 50%
* 75%
* 100%

每个节点都会实时显示：

* ✅ Completed
* ⏳ Remaining

---

## 🔒 Privacy Mode

一键隐藏金额信息。

隐藏内容包括：

* 当前存款
* 剩余金额
* 本月新增
* 今年新增
* 最大单笔
* 平均每月新增
* 历史记录金额
* 趋势图金额
* 目标金额
* 里程碑金额

金额统一显示：

```
¥***
```

但仍保留：

* 完成百分比
* Milestone 状态

适合演示、截图或公开分享。

---

## 💾 数据管理

支持：

### CSV 导出

# 🎨 设计理念

Project Dawn 不希望成为又一个复杂的理财工具。

它更像是一块陪伴用户长期成长的财富仪表盘。

通过：

* 极简设计
* 柔和动态背景
* 可视化目标
* 长期进度追踪

让存钱这件事变得更有仪式感。
